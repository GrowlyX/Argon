# Argon
Server Data Manager which uses Redis for cross-server communications

## Features:
• Redis Support
• Optimized
• Quick
• Free & Open Source
• Semi-Configurable
• Light-weight

## Commands:
• /datadump
• /forceupdate

## Permissions:
• argon.manager - Use argon commands.
